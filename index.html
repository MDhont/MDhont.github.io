--- layout: default --- # [](#header-1)Shuffling pixels Back in the nineties, the artist [Bram DHont](http://kunstkoker.nl/), who also happens to be my father, discovered a strange phenomenon. He ran a Photoshop script on an image with the following logic: ``` original image -> resize 1/2 -> place {top: 0 left: 0} original image -> rotate 90° -> resize 1/2 -> place {top: 0 left: originalImageWidth/2} original image -> rotate 180° -> resize 1/2 -> place {top: originalImageHeigth/2 left: originalImageWidth/2} original image -> rotate 270° -> resize 1/2 -> place {top: 0 left: originalImageWidth/2} ``` The result of running the script once looked like this: |After one round |After 5 rounds | After 10 rounds | |![](/images/twistExample.png)|![](/images/twistExample_2.png)|![](/images/twistExample_3.png)| After five rounds the image contains a lot of tiny sunflowers. After five more rounds the flowers are too small to differentiate and the image looks more like a 70s wallpaper. But, surprise, after twenty five rounds the original image eventually returns. |![](/images/twistExample_4.png)|![](/images/twistExample_5.png)|![](/images/twistExample_6.png)|![](/images/twistExample_7.png)| We were going to have sunflower soup for diner, so what's up with that.
